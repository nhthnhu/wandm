## Features
- Play songs from device's storage
- Browser artists, albums, playlists, folders
- Repeat, shuffle
- Set alarm to stop music
- Search songs
- Add song to favorite playlist and orthers
- Chaneg Light/Dark theme
- Change text size
- Control on notification
